# telegram-bot-news-ai-chat

Бот помогает писать тексты и новости в заданном стиле, хранит персональные настройки пользователя и поддерживает простую админ-панель.

## Возможности

- авторизация пользователей по Telegram userId
- общение с ChatGPT через Telegram
- отдельный prompt для каждого пользователя
- команды `/new`, `/reset`, `/prompt`, `/status`
- глобальные и пользовательские настройки

## Технологии

- Python
